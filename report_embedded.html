TinyPhysics Controls Challenge Report

Overview

This project simulates and evaluates control strategies for the TinyPhysics model using ONNX Runtime. The repository includes:

  • tinyphysics.py: Defines the TinyPhysicsModel and TinyPhysicsSimulator, along with utilities for dataset loading and plotting.
  • eval.py: Script to run sample and batch rollouts, compare test and baseline controllers, collect costs, and generate visual reports.
  • controllers/base_controller.py: Abstract base class defining the controller interface.
  • controllers/my_controller.py: Example proportional controller implementation.
  • controllers/pid_controller.py: PID controller implementation with configurable gains.
  • controllers/zero_controller.py: Controller that outputs zero control (baseline).
  • controllers/__init__.py: Controller package initializer.

Setup and Usage

  1. Install dependencies:
    pip install numpy pandas onnxruntime matplotlib seaborn tqdm
  2. Ensure the TinyPhysics ONNX model is available, e.g.:
    ./models/tinyphysics.onnx
  3. Prepare a directory of CSV files or a URL to a ZIP of CSVs.

Controllers

MyController

Proportional controller using gain * (target - current), clipped to [-2, 2].

PIDController

Standard PID controller with kp, ki, kd gains, maintaining integral and derivative terms.

ZeroController

Zero-output controller for baseline comparison.

Conclusion

This report summarizes the codebase structure, setup instructions, controller implementations, and evaluation results for the TinyPhysics control simulations. The modular design supports easy extension with new controllers and datasets.
